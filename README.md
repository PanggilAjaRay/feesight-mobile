## How to install 🔧
Clone this repository and import into **Android Studio**
```bash
git clone https://github.com/Feesight/Feesight.git
```

## Local Properties Configuration 🧑‍💻
### Keystores:
Create `local.properties` with the following info:
```properties

sdk.dir=<sdk_location>

```
